# Sublime Text 基础文本编辑功能

1. **新建文件** (Ctrl+N)
2. **打开文件** (Ctrl+O)
3. **保存文件** (Ctrl+S) / 另存为 (Ctrl+Shift+S)
4. **关闭文件/标签** (Ctrl+W)
5. **撤销** (Ctrl+Z) / **重做** (Ctrl+Y)
6. **剪切** (Ctrl+X) / **复制** (Ctrl+C) / **粘贴** (Ctrl+V)
7. **全选** (Ctrl+A)
8. **查找** (Ctrl+F) / **替换** (Ctrl+H)
9. **跳转到指定行** (Ctrl+G)
10. **跳转到文件** (Ctrl+P)
11. **多光标编辑** — Ctrl+D 逐个选中
12. **缩进** (Tab) / **反缩进** (Shift+Tab)
13. **注释切换** (Ctrl+/)
14. **移动行** — Ctrl+Shift+上/下 移动整行
15. **复制行** — Ctrl+Shift+D 复制当前行
16. **删除行** — Ctrl+Shift+K 删除当前行
