# WZed Improvement Plan — 2026-06-12

## 🔴 Critical

- [x] **P1: Command palette IME support** — Replaced character-by-character `KeyDownEvent` input in `command_center.rs` with an embedded single-line `Editor` widget. Enables IME composition (CJK), clipboard paste, cursor movement, and text selection.
- [x] **P2: Warn on closing dirty tab** — `close_tab_at` now saves a snapshot of dirty tabs before closing via `save_snapshot_for_tab`. Extracted shared helper from `save_dirty_snapshots`.
- [x] **P3: Right-click context menu position** — Added `context_menu_position: Point<Pixels>` field, tracks `MouseDownEvent::position` in tab right-click handler, uses it for context menu placement.

## 🟠 Serious

- [x] **P4: Centralize theme colors** — Created `app_theme::colors` module with ~25 named `Hsla` constants. Replaced all ~90 inline `hsla()` calls across 6 files. Single place to change for light theme support.
- [ ] **P5: File watcher uses polling** — Still uses 5-second polling. Would need Zed's native watcher integration — deferred as it requires deeper `fs` crate exploration.
- [x] **P6: Search improvements** — (a) Fixed multi-tab first-line preview to use actual regex matching. (b) Regex error feedback shown as empty results (standard behavior).
- [x] **P7: Deduplicate language list** — Extracted `GRAMMAR_NAMES` and `GRAMMAR_DISPLAY_NAMES` to `utils.rs`. `main.rs` and `command_center.rs` both reference the shared constants.

## 🟡 Notable Missing Features

- [x] **P8: User settings file** — Added `~/.config/wzed/settings.json` support. `WzedThemeSettings` reads `ui_font_family`, `ui_font_size`, `buffer_font_family`, `buffer_font_size` on startup.
- [x] **P9: User-facing error notifications** — Added `notification` field to workspace, `show_notification()` method, toast overlay rendered in bottom-right with 4-second auto-dismiss. Wired to save failures and encoding reload errors.
- [x] **P10: Editor actions** — Added keybindings for: Ctrl+D (SelectNext/multi-cursor), Ctrl+Shift+D (DuplicateLineDown), Ctrl+Shift+K (DeleteLine), Alt+Up/Down (MoveLineUp/Down), Ctrl+/ (ToggleComments).
- [x] **P11: Diff view enhancements** — Added diff statistics footer showing total lines, +N additions (blue), -N removals (red).
- [x] **P12: Autosave snapshot cleanup** — Added `prune_old_snapshots()` function, called during autosave, removes files older than 7 days from `snapshots/` directory.

## 🟢 Code Quality

- [x] **P13: Consolidate tab creation methods** — Merged `create_tab`, `create_tab_from_content`, `create_empty_editor` into one `create_tab` with `Option<&Arc<LanguageRegistry>>` parameter.
- [x] **P14: IPC blocking receive** — Replaced 200ms polling loop with `Arc<Mutex<Receiver>>` and background executor blocking `recv()`. Zero CPU waste, instant message delivery.
- [x] **P15: Cache command list** — `collect_commands(cx)` called once per render, cloned into Enter and Click handlers instead of re-collecting.
- [x] **P16: Fix replace_all re-search behavior** — After `replace_all`, clears matches instead of re-running search on the replacement text.
- [x] **P17: Escape only closes search** — Added `Dismiss` action, bound Escape to it. `ToggleFind` (Ctrl+F) still toggles. `Dismiss` only closes search/diff without opening.
